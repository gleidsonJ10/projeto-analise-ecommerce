# 📊 Projeto de Análise de Dados - E-commerce

Este projeto tem como objetivo analisar dados de um e-commerce utilizando Python, Pandas, Matplotlib e Seaborn.

## 📌 Tecnologias utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Análises realizadas
Foram criados os seguintes gráficos:

- Histograma (Distribuição de preços)
- Gráfico de dispersão (Preço x Nota)
- Mapa de calor (Correlação entre variáveis)
- Gráfico de barras (Top marcas)
- Gráfico de pizza (Distribuição por temporada)
- Gráfico de densidade (Distribuição dos preços)
- Gráfico de regressão (Preço x Quantidade vendida)

## 📁 Dataset
Arquivo: `ecommerce_estatistica.csv`

## 🎯 Objetivo
Extrair insights dos dados para auxiliar na tomada de decisão.
